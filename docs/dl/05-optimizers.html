Optimizers

Optimizers

An optimizer is a function that takes a tensor and return another tensor that's derived from the class rc.optimizer. Its interface is exactly the same as the ones in differentiable functions. Only this time the minimize target doesn't need to be specified.

Predefined Optimizers

Name Arguments
sgd lr=0.001, momentum=0.0, decay=0.0, nesterov=False
rmsprop lr=0.001, rho=0.9, epsilon=1e-9, decay=0.0
